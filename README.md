# Student Grade Calculator

## 📌 Project Overview
This Java application calculates a student's total marks, average percentage, and grade based on marks entered for multiple subjects.

## 🚀 Features
- Accepts marks for 6 subjects.
- Validates marks (0–100).
- Calculates total marks.
- Calculates average percentage.
- Assigns grades based on average percentage.
- Displays the final result.

## 📊 Grade Criteria
- A Grade: 80% and above
- B Grade: 60% to 79%
- C Grade: 40% to 59%
- Fail: Below 40%

## ▶️ How to Run
1. Open the project in VS Code.
2. Compile the program:
   ```
   javac DecodeLabs_Java_P2.java
   ```
3. Run the program:
   ```
   java DecodeLabs_Java_P2
   ```
4. Enter marks for each subject.
5. View the total marks, average percentage, and grade.
